MAME4droid CHANGELOG

DEVELOPMENT:
  • 2024-09-09 Version 1.16
    Updated to MAME 0.269.
  • 2024-08-05 Version 1.15
    Updated to MAME 0.268.
  • 2024-07-03 Version 1.14
    Updated to MAME 0.267.
  • 2024-05-30 Version 1.13
    Updated to MAME 0.266.
  • 2024-04-26 Version 1.12
    Updated to MAME 0.265. Improved MAME touch UI. Some font tweaking and frontend resolution bumped.
  • 2024-03-26 Version 1.11
    Updated to MAME 0.264. Improved haptics. Added BSP-D8 controller support. Added some new nogeo shaders.
  • 2024-02-28 Version 1.10
    Updated to MAME 0.263. Improved SAF support for MESS software media. Added right stick and analog triggers support to generic controllers
  • 2024-02-09 Version 1.9
    Added media scraping thanks to Arcade Database by motoschifo. Added MAME meta data resources thanks to Arcade History, MAME info and progetto-SNAPS
  • 2024-01-31 Version 1.8
    Updated to MAME 0.262. Added two small additional buttons so they can be configured through the input macro plugin for fighting games. Added touch controller transparency selectable
  • 2024-01-19 Version 1.7
    Implemented new OpenGLES 3.2 faster renderer with advanced postprocessing effects using shaders. Added speed hacks to improve performance in games like K.Instinct. Fixed video stuttering on some occasions. Improved audio latency.
  • 2024-01-03 Version 1.6
    Fixed N.System22 driver graphical glitches on ARM64. Emulated and frontend resolution independently selectable. Improved AndroidTV performance.
  • 2023-12-29 Version 1.5
    Added option to set the entire app to a fixed orientation (like landscape). Bug fixes.
  • 2023-12-27 Version 1.4
    Fixed E,F buttons not assigned. Added combo up/down+a to fast scroll in game list. Performance improvements.
  • 2023-12-21 Version 1.3
    Improved touch lightgun so i can play t.crisis :). Bug fixes
  • 2023-12-20 Version 1.2
    Fixed coin button does not work in several games. Added option in menu to improve performance in some 3D games like System 22.
  • 2023-12-19 Version 1.1
    Bug fixes. Added combo start+a+b as service mode and coin+a+b as soft reset.
  • 2023-12-17 Version 1.0
    First version.
